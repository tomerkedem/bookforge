# Logistic Regression

While linear regression predicts a numerical value, logistic regression is used for classification.

The goal is to estimate the probability that a data point belongs to a certain category.

The model returns a value between 0 and 1 using the sigmoid function.

So the result represents a probability.

The model doesn't just give an answer,

but also expresses how confident it is in it.

The model transforms a numerical prediction into a probability-based decision

**Using a decision threshold, probability can be turned into a decision:**

Above a certain value the result will be classified as one category, and below it as another category.
