# Aprendizaje Supervisado

En el aprendizaje supervisado, el modelo aprende de datos que tienen respuestas conocidas de antemano.

Es decir, cada ejemplo viene con una etiqueta que representa el resultado correcto.

El modelo aprende a identificar la relación entre los datos y el resultado,

y luego usa lo que aprendió para predecir resultados para datos nuevos.

<img src="/ml-lesson-2-summary/assets/image-08.png" alt="image-08.png" width="710" height="205" />

**Proceso de Trabajo**

El modelo entrena con datos etiquetados y aprende a identificar patrones.

Luego recibe datos nuevos sin etiquetas,

y da una predicción según lo que aprendió.

La salida es el resultado que el modelo estima.

**Ejemplo**

Por ejemplo, identificar un tipo de fruta de una imagen por sus características.

El modelo no recibe reglas predefinidas, sino que las aprende de los datos.
