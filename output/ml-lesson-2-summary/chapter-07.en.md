# Types of Learning in Machine Learning

1. **Supervised Learning**

In supervised learning there is data with known answers in advance,

and the model learns to predict the correct result.

**In this mode there are two types of problems:**

- When the result is a number, we predict a continuous value

- When the result is a category, we classify into groups

2. **Unsupervised Learning**

In unsupervised learning there are no known answers in advance,

and the model tries to find structures and patterns in the data by itself.

**In this mode there are two main types:**

- Clustering data into groups by similarity

- Identifying relationships between items that appear together

**Supervised Learning**

In supervised learning, the model learns from data that has known answers in advance.

The data comes with labels -
meaning, each example has a correct answer

The algorithm learns to identify the relationship between the data and the result,
and then uses what it learned to predict results for new data

Work process:

Start with labeled data
The model trains on them and learns the patterns

Then receive new data without labels
and the model gives a prediction or classification

The output is the result the model estimates -
for example, identifying a type of fruit from an image
