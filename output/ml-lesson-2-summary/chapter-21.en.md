# Neuron

A neuron is a basic computational unit that receives multiple inputs and produces a single output.

Each input is multiplied by a weight that expresses its importance, and then all values are summed together.

The result passes through a function that decides what the output will be.

You can think of it as a decision-making process:

Multiple factors influence together, but not all to the same degree.

<img src="/ml-lesson-2-summary/assets/image-21.png" alt="image-21.png" width="671" height="352" />

The output is obtained from the combination of all inputs.
