# Unsupervised Learning - Process

The data is fed to the algorithm without prior guidance.
The algorithm identifies internal structure in the data, and creates a new representation of it.

<img src="/ml-lesson-2-summary/assets/image-16.png" alt="image-16.png" width="605" height="250" />

The result can be a division into similar groups or discovery of relationships between data.

**Unsupervised Learning** focuses on discovering patterns and structure in unlabeled data using Clustering, Association, and Dimensionality Reduction

• **Clustering** allows dividing data into groups by similarity.
For example: dividing customers into groups by purchasing behavior.

• **Association** allows identifying relationships between variables.
For example: customers who bought bread also buy milk.

• **Dimensionality Reduction** allows reducing the number of variables.
For example: reducing the amount of information in an image while maintaining quality.

**The model doesn't receive answers, but discovers patterns within the data**
