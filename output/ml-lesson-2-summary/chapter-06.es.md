# Cómo Funciona el Proceso de Machine Learning en la Práctica

Machine Learning es un proceso en el cual el sistema aprende de los datos,

para hacer predicciones sobre datos nuevos.

<img src="/ml-lesson-2-summary/assets/image-06.png" alt="image-06.png" width="710" height="293" />

**El proceso consta de cuatro etapas principales:**

• Comenzar con la recolección de datos

• Dividir los datos en entrenamiento y prueba

• Entrenar un modelo para identificar patrones en los datos

• Usar el modelo para predecir resultados para datos nuevos

**La clave para entender todo el proceso es esta:**
**El modelo no recibe reglas predefinidas, sino que las aprende de los datos.**
