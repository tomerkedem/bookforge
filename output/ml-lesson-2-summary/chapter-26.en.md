# What You Need to Know as an AI Engineer

- You don't need to be a Machine Learning expert.

- You need to understand the concepts of Supervised Learning, Unsupervised Learning, and Deep Learning.

- You need to know how to deploy and run pre-trained models in a production environment.

- You need to understand the ML lifecycle:

<div style="text-align: center">

Train → Measure → Deploy → Predict

</div>

- You need to turn the model into a Production-ready application: 
Scalable, stable, cost-efficient, meets performance requirements, and monitorable.

- The model is the engine, but you need to build the whole car.
