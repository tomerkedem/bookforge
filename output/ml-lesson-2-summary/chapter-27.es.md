# NumPy: Trabajo Eficiente con Datos Numéricos

NumPy es la biblioteca central de Python para trabajar con datos numéricos. Proporciona una forma significativamente más rápida y eficiente de manejar datos en comparación con el uso de listas regulares de Python.

<img src="/ml-lesson-2-summary/assets/image-32.png" alt="image-32.png" width="646" height="294" />

**Capacidades Principales y Ventajas Tecnológicas**

• **Alto Rendimiento:** El núcleo de la biblioteca está escrito principalmente en C. Este hecho permite realizar cálculos complejos a una velocidad excepcional.

• **Operaciones Vectorizadas:** Las operaciones matemáticas se pueden realizar en arrays completos sin necesidad de escribir bucles lentos y explícitos. Este enfoque hace que el código sea más simple de mantener y mucho más rápido de ejecutar.

• **Mecanismo de Broadcasting:** Permite realizar operaciones aritméticas entre arrays de diferentes tamaños. El mecanismo "ajusta" los arrays entre sí automáticamente para el cálculo.

• **Amplio Kit de Herramientas Matemáticas:** NumPy proporciona un rico conjunto de funciones integradas. Estas incluyen soluciones para álgebra lineal, transformaciones complejas y generación de números aleatorios.
