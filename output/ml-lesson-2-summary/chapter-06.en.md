# How the Machine Learning Process Works in Practice

Machine Learning is a process in which the system learns from data,

to make predictions on new data.

<img src="/ml-lesson-2-summary/assets/image-06.png" alt="image-06.png" width="710" height="293" />

**The process consists of four main stages:**

• Start with data collection

• Split the data into training and testing

• Train a model to identify patterns in the data

• Use the model to predict results for new data

**The key to understanding the entire process is this:**
**The model does not receive pre-made rules, but learns them from the data.**
