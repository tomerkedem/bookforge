# Regresión Lineal (Linear Regression)

Un modelo cuyo objetivo es aprender una relación entre variables,
para predecir un valor numérico.

<img src="/ml-lesson-2-summary/assets/image-10.png" alt="image-10.png" width="531" height="346" />

La línea roja representa el modelo,
El modelo no pasa por todos los puntos,
sino que busca la línea que mejor se ajusta.

El modelo no intenta acertar cada punto,
sino ajustarse a la tendencia general.

**Cómo Medir el Error**

Cada punto tiene una distancia desde la línea,
esta distancia se llama error.

El modelo busca una línea que minimice la suma de errores al cuadrado.
