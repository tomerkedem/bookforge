# Tipos de Aprendizaje en Machine Learning

1. **Aprendizaje Supervisado**

En el aprendizaje supervisado hay datos con respuestas conocidas de antemano,

y el modelo aprende a predecir el resultado correcto.

**En este modo hay dos tipos de problemas:**

- Cuando el resultado es un número, predecimos un valor continuo

- Cuando el resultado es una categoría, clasificamos en grupos

2. **Aprendizaje No Supervisado**

En el aprendizaje no supervisado no hay respuestas conocidas de antemano,

y el modelo intenta encontrar estructuras y patrones en los datos por sí mismo.

**En este modo hay dos tipos principales:**

- Agrupar datos en grupos por similitud

- Identificar relaciones entre elementos que aparecen juntos

**Aprendizaje Supervisado**

En el aprendizaje supervisado, el modelo aprende de datos que tienen respuestas conocidas de antemano.

Los datos vienen con etiquetas -
es decir, cada ejemplo tiene una respuesta correcta

El algoritmo aprende a identificar la relación entre los datos y el resultado,
y luego usa lo que aprendió para predecir resultados para datos nuevos

Proceso de trabajo:

Comenzar con datos etiquetados
El modelo entrena con ellos y aprende los patrones

Luego recibir datos nuevos sin etiquetas
y el modelo da una predicción o clasificación

La salida es el resultado que el modelo estima -
por ejemplo, identificar un tipo de fruta de una imagen
