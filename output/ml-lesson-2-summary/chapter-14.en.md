# Gradient Boosting (XGBoost)

An algorithm based on decision trees built one after the other.

<img src="/ml-lesson-2-summary/assets/image-14.png" alt="image-14.png" width="614" height="390" />

At each step a new tree is built,
that focuses on the errors of the previous tree.

This way the model improves gradually,
and handles difficult cases better.

**A decision tree solves one problem, a random forest votes, and gradient boosting learns from mistakes.**
