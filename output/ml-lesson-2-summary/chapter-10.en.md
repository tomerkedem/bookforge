# Linear Regression

A model whose goal is to learn a relationship between variables,
to predict a numerical value.

<img src="/ml-lesson-2-summary/assets/image-10.png" alt="image-10.png" width="531" height="346" />

The red line represents the model,
The model doesn't pass through all the points,
but looks for the line that fits best.

The model doesn't try to hit every point,
but to fit the general trend.

**How to Measure Error**

Each point has a distance from the line,
this distance is called error.

The model looks for a line that minimizes the sum of squared errors.
