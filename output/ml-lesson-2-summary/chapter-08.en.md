# Supervised Learning

In supervised learning, the model learns from data that has known answers in advance.

That is, each example comes with a label representing the correct result.

The model learns to identify the relationship between the data and the result,

and then uses what it learned to predict results for new data.

<img src="/ml-lesson-2-summary/assets/image-08.png" alt="image-08.png" width="710" height="205" />

**Work Process**

The model trains on labeled data and learns to identify patterns.

Then it receives new data without labels,

and gives a prediction according to what it learned.

The output is the result the model estimates.

**Example**

For example, identifying a type of fruit from an image by characteristics.

The model does not receive pre-made rules, but learns them from the data.
