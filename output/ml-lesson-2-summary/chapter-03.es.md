# ¿Por qué usar Machine Learning?

Con Machine Learning, las computadoras pueden aprender de los datos y adaptar sus acciones según la información que reciben.

En lugar de definir todas las reglas de antemano, el sistema aprende patrones de grandes cantidades de datos y mejora su rendimiento con el tiempo.

<img src="/ml-lesson-2-summary/assets/image-03.png" alt="image-03.png" width="709" height="220" />

El objetivo es ilustrar que el comportamiento del sistema se deriva de los datos y no de reglas fijas.

Machine Learning es la capacidad de aprender de los datos automáticamente, y luego lidiar con casos nuevos que no se habían visto antes.

Por ejemplo, si se entrena un modelo para distinguir entre imágenes de gatos y perros, luego se le puede mostrar una nueva imagen de un perro que no apareció en los datos de entrenamiento, y podrá identificarla basándose en las características que aprendió.
