# Why Use Machine Learning

With Machine Learning, computers can learn from data and adapt their actions according to the information they receive.

Instead of defining all the rules in advance, the system learns patterns from large amounts of data and improves its performance over time.

<img src="/ml-lesson-2-summary/assets/image-03.png" alt="image-03.png" width="709" height="220" />

The goal is to illustrate that the system's behavior is derived from the data and not from fixed rules.

Machine Learning is the ability to learn from data automatically, and then deal with new cases that were not seen before.

For example, if a model is trained to distinguish between pictures of cats and dogs, it can then be shown a new picture of a dog that did not appear in the training data, and it will be able to identify it based on the features it learned.
