# Reinforcement Learning

<img src="/ml-lesson-2-summary/assets/image-19.png" alt="image-19.png" width="468" height="304" />

Reinforcement Learning is an approach in which an agent learns through interaction with an environment.

Unlike supervised and unsupervised learning, the learning here is based on experience and trial.

In the learning process:
- The **Agent** observes the **State** of the environment
- Performs an **Action**
- Receives a **Reward** (or penalty)

Based on this information, the agent develops a **Policy** - a consistent way of action in each situation to maximize the long-term expected reward.

Learning happens through many trials, in which the model recognizes repeated patterns - what worked? What failed?
Gradually, the agent learns to choose the actions that lead to maximum benefit.

Used in games, autonomous driving, control systems, and more.

**Reinforcement Learning: learning through trials and reward for actions**
