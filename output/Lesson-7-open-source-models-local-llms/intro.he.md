# פתיחה: מהרצת API להבנה אמיתית של מודלים

עד עכשיו עבדנו עם מודלים בעיקר כעם שירות חיצוני: שולחים prompt, מקבלים תשובה, ומשלבים את התוצאה באפליקציה. זו דרך מצוינת להתחיל, אבל כאשר מערכת AI הופכת למערכת אמיתית, המודל כבר אינו רק כתובת API. הוא הופך לרכיב הנדסי שצריך לבחור, לבדוק, למדוד ולנהל.

כאן מתחיל עולם המודלים הפתוחים.

היתרון ברור: אפשר להוריד מודלים, להריץ אותם מקומית, לבדוק כמה חלופות, לשמור מידע רגיש בתוך הארגון, לחסוך חלק מעלויות השימוש, ולהקטין תלות בספק חיצוני. אבל יחד עם היתרון מגיעה גם אחריות. לא כל מודל שאפשר להוריד הוא באמת מודל פתוח במובן המלא. במקרים רבים מקבלים בעיקר את המשקלים של המודל, כלומר את ה weights, בלי לקבל את כל תהליך האימון, הדאטה, הקוד והלוגים שמסבירים איך הוא נבנה. לכן כבר בתחילת העבודה עם מודלים כאלה צריך לבדוק מי פרסם אותם, מה הרישיון מאפשר, ומה באמת פתוח בהם.

המרכז הגדול ביותר לעבודה כזו הוא Hugging Face. שם אפשר למצוא מודלים, datasets, אפליקציות הדגמה, דוגמאות קוד, Model Cards, רישיונות, קבצים ונתוני שימוש. אבל Hugging Face אינו רק מקום שממנו מורידים מודלים. הוא כלי עבודה לבחירה מושכלת: מחפשים מודל לפי משימה, קוראים את התיעוד, בודקים את הרישיון, מבינים את דרישות ההרצה, ורק אז מריצים ניסוי אמיתי.

לאחר החלק העיוני עוברים לעבודה מעשית. נקים פרויקט פייתון מסודר, ניצור סביבת עבודה נפרדת, נתקין transformers, torch ו accelerate, ונלמד איך להריץ מודלים מתוך קוד. נתחיל ממודל סיווג רגשות, שבו הקלט הוא טקסט והפלט הוא תווית כמו Positive או Negative. לאחר מכן נעבור למודל יוצר טקסט כמו Qwen, שמקבל prompt ומייצר תשובה חופשית.

בהמשך נכיר את Ollama, כלי שמפשט מאוד הרצה מקומית של מודלים. במקום לנהל בעצמנו את כל קובצי המודל וההגדרות, Ollama מאפשר להוריד מודל, להריץ אותו על המחשב, לחשוף אותו דרך API מקומי, ולחבר אליו סקריפטים, אפליקציות או Agents.

המטרה היא לא רק לדעת להריץ מודל. המטרה היא להבין איך מתייחסים למודל כרכיב תוכנה רציני: בוחרים אותו לפי צורך, בודקים אותו על דוגמאות אמיתיות, מבינים את מגבלותיו, שולטים בגרסה שלו, ומחליטים מתי להריץ אותו מקומית ומתי עדיף להשתמש בשירות חיצוני.

זה המעבר המרכזי: ממודל כקופסה מרוחקת, למודל כרכיב הנדסי שאפשר להבין, לבדוק ולשלב בצורה אחראית במערכת אמיתית.
