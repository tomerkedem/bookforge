# קריאת קבצי MD בעברית

## תיאור
קריאה נכונה של קבצי MD בעברית עם שמירה על מבנה מלא.

## מתי להשתמש
כשצריך לקרוא קובץ chapter-XX.he.md או chapter-XX.en.md
ולחלץ ממנו כותרות, תוכן, ורשימות.

## שלבים

1. קרא את הקובץ עם encoding UTF-8
2. זהה כותרות לפי # ו-##
3. שמור על כיוון RTL בטקסט עברי
4. זהה רשימות ממוספרות ורשימות נקודות
